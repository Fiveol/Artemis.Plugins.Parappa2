# A plugin for Artemis to get game state data from Parappa the Rapper 2!

## Needs a USA copy of Parappa the Rapper 2 and the latest PCSX2 Nightly

### Exposes: Current rank

Setup:

Enable PINE in PCSX2 Advanced Settings
